🏨 Hotel Booking Chatbot
A conversational chatbot built using Google Dialogflow and deployed as a web application. The bot assists users in booking hotel rooms by guiding them through room selection, pricing, availability, and booking confirmation.

🎯 Project Overview
This project was developed as part of the Cloud Computing & AI domain during my internship at Corizo Edutech. The chatbot simulates a real-world hotel booking experience through natural language conversation.

✨ Features

🛏️ Room type selection (Classic, Duplex, Suite)
💰 Real-time pricing information
📅 Check-in date and duration of stay
✅ Booking confirmation flow
🌐 Web-based interface using Dialogflow Messenger


🛠️ Technologies Used
TechnologyPurposeGoogle DialogflowNLP & Chatbot EngineHTML / CSSFrontend Web InterfaceAWS LexAlternative deploymentJavaScriptDialogflow Messenger integration

🚀 How It Works

User opens the web page
Dialogflow Messenger widget loads in the browser
User types their query (e.g., "I want to book a room")
Bot identifies the intent and collects required information via slot-filling
Bot confirms booking details and provides pricing

📸 Sample Conversation Flow
User  → "I want to book a hotel room"
Bot   → "Welcome! What type of room would you like? Classic, Duplex, or Suite?"
User  → "Classic room"
Bot   → "Classic room costs ₹2,500/night. How many nights would you like to stay?"
User  → "3 nights"
Bot   → "Your total comes to ₹7,500. Shall I confirm your booking?"
User  → "Yes"
Bot   → "Your booking is confirmed! Thank you for choosing us."
